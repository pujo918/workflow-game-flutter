Pengenalan Teknologi & Fitur Utama
Pondasi pengembangan game edukasi KotoCraft

KotoCraft adalah game edukasi interaktif RPG 2D yang dirancang khusus untuk mempermudah pembelajaran aksara Jepang (Hiragana dan Katakana) serta kosakata dasarnya. Game ini memadukan keseruan elemen petualangan RPG dengan pembelajaran bahasa Jepang yang sistematis.

Teknologi yang Digunakan

📱
Flutter Framework
Pondasi utama aplikasi. Framework UI buatan Google ini bertugas menangani struktur halaman non-game (menu utama, pengaturan, kamus, whiteboard menulis) secara cepat dan menghasilkan performa antar-muka yang sangat mulus di berbagai platform dengan satu basis kode.
🎮
Flame Game Engine
Engine game modular 2D yang dibangun di atas Flutter. Flame bertugas menangani rendering grafis, loading tilemap peta (TMX), logika game loop, deteksi tabrakan (collision rintangan duri & huruf melayang), kamera, serta pergerakan karakter utama.
🎯
Dart Language
Bahasa pemrograman utama di balik KotoCraft. Dart menawarkan kecepatan eksekusi tinggi karena dikompilasi langsung ke kode mesin native (AOT) serta memiliki fitur Hot Reload yang mempercepat proses pengembangan logika game secara instan.

Keuntungan & Kerugian Teknologi

✔ Keuntungan
  • Single Codebase: Aplikasi dapat dirilis ke Android, iOS, dan Web dari satu kode yang sama, menghemat waktu pengembangan.
  • Integrasi UI Sangat Mudah: Sangat mudah menyisipkan menu UI standar Flutter (seperti whiteboard coretan dan TTS) langsung ke dalam kanvas game Flame.
  • Proses Dev Cepat: Fitur Hot Reload/Restart mempermudah iterasi desain level dan perbaikan bug tanpa perlu menunggu kompilasi ulang yang lama.
  • Performa Lancar: Kompilasi native Dart dan arsitektur Flame menjamin game berjalan mulus hingga 60 FPS pada perangkat seluler.
✘ Kerugian
  • Ukuran File (APK) Lebih Besar: Engine Flutter dan aset pendukung yang dikemas menyebabkan ukuran file awal aplikasi menjadi sedikit lebih besar.
  • Kurang Cocok untuk Game 3D: Flame Engine khusus dirancang untuk game 2D. Sangat sulit dan tidak disarankan untuk mengembangkan game 3D kompleks di sini.
  • Ekosistem Game Terbatas: Pustaka plugin pihak ketiga khusus game tidak sebanyak engine khusus game lain (seperti Unity atau Godot).

Alur Kerja Game (Game Workflow) secara Sederhana

Bagi pemain, KotoCraft menawarkan alur bermain yang sangat ramah pengguna dan terstruktur:

  1. Belajar Teori di Kamus: Sebelum mulai petualangan, pemain direkomendasikan masuk ke menu Kamus Materi. Di sini mereka belajar cara membaca huruf, mendengarkan pengucapan (TTS), dan melatih tulisan tangan (coretan jari). Setiap huruf yang selesai dipelajari memberikan hadiah XP.
  2. Memilih Level Petualangan: Pemain masuk ke menu Start/Lanjutkan. Pemain memilih level (dari Level 1 hingga 10). Level 1 terbuka di awal, sedangkan level berikutnya terkunci dan hanya terbuka setelah level sebelumnya berhasil diselesaikan.
  3. Memilih Mode Kesulitan: Pada Level 1 hingga 4, pemain dapat memilih mode Easy (fokus belajar tanpa rintangan) atau Normal (dilengkapi rintangan duri yang menantang kelincahan pemain). Level 5 ke atas dijalankan dalam mode Easy.
  4. Eksplorasi & Berburu Huruf: Di dalam game RPG 2D, pemain mengendalikan karakter berjalan mengeksplorasi peta untuk mencari huruf-huruf Jepang melayang.
  5. Kuis Kejutan (Instan): Ketika karakter menyentuh huruf melayang, game otomatis dijeda, dan muncul kuis pilihan ganda untuk menebak romaji huruf tersebut. Jika menjawab dengan benar, pemain mendapat +10 XP. Jika salah, jumlah kesalahan bertambah dan nyawa (HP) bisa berkurang jika bermain di mode Normal.
  6. Evaluasi Akhir Level: Setelah mengumpulkan seluruh huruf di peta, level selesai. Pemain diberikan penilaian berupa bintang (1 sampai 3 bintang) berdasarkan jumlah kesalahan menjawab. Progres level dan total XP yang terkumpul akan langsung disimpan secara permanen.

Detail Fitur Unggulan Aplikasi

📝 Kamus Interaktif & Fitur Coret Huruf (Whiteboard)
Ini adalah modul belajar mandiri yang sangat rinci:
  • Animasi Stroke Order (Urutan Goresan): Setiap huruf dilengkapi panduan visual urutan penulisan yang benar dari goresan pertama hingga terakhir agar pengguna belajar menulis dengan presisi, bukan sekadar menggambar bebas.
  • Kanvas Coretan (Interactive Canvas): Pengguna dapat menulis langsung dengan jari atau stylus di atas kanvas digital yang disediakan untuk mempraktikkan tulisan mereka secara langsung.
  • Text-to-Speech (TTS) Pengucapan: Dilengkapi tombol suara native Jepang untuk melatih pendengaran (listening) dan pengucapan (pronunciation) secara akurat.
🗺️ Eksplorasi RPG 2D Top-Down
Peta petualangan 2D interaktif berbasis tilemap (TMX) di mana pemain mengendalikan karakter utama dengan D-Pad virtual di layar atau tombol WASD/Arah di keyboard desktop. Pemain dapat menikmati dunia game sambil mencari huruf yang tersembunyi di berbagai sudut peta.
🔄 Kuis Pilihan Ganda Dinamis & Shuffle Mode
Untuk menguji pemahaman, kuis romaji muncul secara otomatis.
  • Sistem Pengacakan Jawaban: Pilihan jawaban diacak setiap kali kuis dipicu, menghindari sistem hafalan letak tombol.
  • Level 9 & 10 (Katakana Acak): Khusus level ini, huruf Katakana yang disebar di peta diacak secara total dari bank data Katakana setiap kali pemain memulai ulang game, memberikan tantangan belajar baru yang dinamis.
⚠️ Sistem HP, Rintangan, & Mode Kesulitan (Normal Mode)
Memberikan bumbu tantangan RPG aksi di mana peta level memiliki jebakan duri. Jika karakter menabrak rintangan ini, nyawa (HP) pemain akan berkurang. Jika HP habis, game berakhir (Game Over). Fitur ini melatih kecepatan reaksi motorik di samping kemampuan akademis.
⚡ Sistem XP Terpadu & Level Unlock
Poin pengalaman (XP) yang diperoleh dari berpetualang (+10 XP per jawaban kuis benar) dan belajar di kamus (+5 XP per huruf baru) digabungkan menjadi satu nilai kumulatif di profil pemain. Sistem otomatis melacak level mana saja yang sudah berhasil diselesaikan (unlock level progresif).
💾 Persistensi Data (Autosave)
Menggunakan SharedPreferences untuk menyimpan seluruh riwayat kemajuan pemain secara otomatis (level yang terbuka, jumlah bintang per level, akumulasi XP, status bab kamus yang dibaca) sehingga progres tidak akan hilang saat aplikasi ditutup.

Alur Gameplay Utama
Mekanisme eksplorasi 2D top-down dengan kuis evaluasi dinamis

Mekanisme gameplay menggabungkan eksplorasi dunia 2D tipe top-down menggunakan engine Flame dengan kuis evaluasi dinamis ketika item huruf Jepang ditemukan oleh pemain.

Langkah Operasional:

  1. Memilih Level — Dari beranda, tekan START/LANJUTKAN untuk membuka grid wilayah. Level 1 terbuka secara default, level selanjutnya dikunci berurutan.
  2. Memilih Kesulitan — Level 1-4 memiliki pilihan Easy (tanpa rintangan) atau Normal (dengan duri/traps). Level 5-10 hanya Easy.
  3. Eksplorasi Peta — Gerakkan karakter menggunakan joystick virtual atau keyboard WASD untuk mencari huruf Jepang yang melayang.
  4. Kuis Otomatis — Saat menyentuh huruf, game pause dan kuis romaji 4 pilihan muncul. Benar = +10 XP, salah = +1 Mistake.
  5. Sistem HP (Normal) — Menabrak duri mengurangi HP. HP habis = Game Over.
  6. Bintang & Selesai — Semua huruf terkumpul = level selesai. Bintang berdasarkan kesalahan:
3 Bintang — 0 salah 2 Bintang — 1-2 salah 1 Bintang — 3+ salah
Penyimpanan Progres Kelulusan level disimpan permanen di SharedPreferences via SaveService. Data: level selesai, bintang, total XP, dan karakter avatar.
Sistem XP Terpadu XP dari game (+10 per jawaban benar) dan XP dari kamus materi (+5 per huruf dipelajari) digabungkan menjadi satu nilai kumulatif yang ditampilkan di beranda.

Alur Pembelajaran Materi
Modul kamus teori dengan 2 bab bertahap dan sistem unlock

Modul Kamus Materi memberikan landasan teori sebelum pemain terjun ke level gameplay. Alur materi disusun berurutan dengan sistem unlock bertahap.

Pengenalan karakter dasar Jepang melalui tiga tab interaktif:

  1. Tab Hiragana & Katakana — Grid visual huruf. Klik untuk membuka pop-up: TTS pengucapan, animasi stroke order, dan canvas latihan menulis. Setiap huruf baru = +5 XP.
  2. Tab Kosakata — Kartu kata benda/kerja dengan filter kategori dan audio TTS.
  3. Penyelesaian — Tombol SELANJUTNYA membuka galeri materi tambahan (zoomable). Menutup galeri memanggil completeLevel('materi_1') yang membuka Bab 2.

Menggunakan ChapterContentViewerScreen dengan 4 slide gambar materi. Navigasi via tombol LANJUT/KEMBALI. Di slide terakhir, tombol SELESAI memanggil completeLevel('materi_2').

Persistensi Progres Belajar Progres belajar kamus (huruf yang sudah dipelajari) disimpan permanen di SharedPreferences via LearningProvider. Saat aplikasi dibuka kembali, semua progres tetap ada. XP kamus terjumlah dengan XP gameplay.
